# Home Assistant Blueprint: Bewegungsgesteuertes Licht (nur automatische Abschaltung)

Schaltet ein Licht bei Bewegung ein, aber **nur wenn es vorher aus war**.  
Das Licht wird nach einer Verzögerung **nur ausgeschaltet**, wenn es durch Bewegung eingeschaltet wurde.

✅ Manuelle Aktivierung bleibt unberührt  
✅ Funktioniert mit optionalem Helligkeitssensor  
✅ Verwendet einen `input_boolean` zur Erkennung
